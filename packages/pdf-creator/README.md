# @jmt-code/pdf-creator

A library for generating PDF documents.

## Installation

```bash
npm install @jmt-code/pdf-creator
```

## Usage

```typescript
import { PdfCreator } from '@jmt-code/pdf-creator';

const pdf = new PdfCreator();
```
